## About

**Projexa** is a project-based mentoring system designed for educational institutions. It connects students with mentors, manages team projects, tracks evaluations, and streamlines academic project workflows â€” all from your phone.

## Download

Head to the [**Releases**](../../releases) page and download the latest `.apk` file.

> **Note:** Since this is not distributed via the Play Store, you'll need to allow installation from unknown sources on your Android device.

## Installation

1. Download the latest `.apk` from [Releases](../../releases)
2. Open the file on your Android device
3. If prompted, allow installation from unknown sources
4. Install and open the app
5. Log in with your institutional credentials
